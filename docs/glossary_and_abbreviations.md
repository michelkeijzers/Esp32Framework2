# GLOSSARY

| Term          | Definition                                                                                                                                          |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| Master (Node) | The primary node in a cluster that manages the overall operation and coordination of the cluster. There is only one Master Node in one application. |
| Application   | A master node and zero or more slave nodes.                                                                                                         |
| Bridge        | The connection between the Master task and slave tasks.                                                                                             |
| Cluster       | A group of nodes that work together to perform specific tasks or functions.                                                                         |
| Node          | A device that is part of a cluster and performs specific tasks or functions. One node is one ESP.                                                   |
| Slave (Node)  | A node in a cluster that is managed by the App (Node) and performs specific tasks or functions. One slave node is one ESP.                          |
